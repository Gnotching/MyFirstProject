APA ITU SHORT MESSAGE ?

Halo sobat semua,

Perkenalkan saya adalah pembuat artikel ini, kalian bisa sebut saya kak noting.Kali ini kak noting akan membahas tentang catatan pribadi kak noting yang berjudul Short Message.Artikel ini adalah artikel pertama kak noting di website Seragam biru.Kalau kalian ingin kepoin profil kak noting kalian bisa mengunjungi website Profil kak noting, disana banyak penjelasan tentang kak noting.Oke, langsung aja kita ke penjelasannya secara simple.

Bila diartikan satu-satu, short artinya pendek dan message artinya pesan.Bila disatukan kata perkata menjadi pesan pendek.Tetapi, secara pengertian dalam bahasa inggris "Short Message is a brief text message sent to or from a mobile phone subsciber through the Short Message Service (SMS)".Bila diartikan, pesan singkat merupakan informasi atau pemberitahuan yang dikirimkan seseorang kepada orang lain baik dengan menuliskannya pada kertas ataupun mengirimnya melalui SMS.




Purpose of Short Message

To send an important message or information to other people, friend or family

tujuan short message



Bagaimana cara kita tahu bahwa text tersebut merupakan text Short Message ?Perlu diketahui bahwa text tersebut memiliki beberapa ciri-ciri atau characteristic-nya, diantaranya :

  • Part of Short Message :
    1. Sender (pengirim) : people who send some message
    2. Receiver (penerima) : people who receive or get some message from sender
    3. Subject (perihal/maksud/tujuan) : contain some request or purpose from an sender to tell/inform receiver
  • You must use short text for make Short Message(Kamu harus menggunakan text singkat)
  • Bahasa yang digunakan bebas dan informal

Biasanya Short Message sering kita jumpai apabila ada keluarga atau teman, baik teman sekolah/teman kantor yang ingin menjumpai kita tetapi tidak sempat untuk bertemu.Maka mereka akan mengirimkan pesan yang singkat kepadamu bila ada keperluan mendadak seperti; mendadak meeting, ibu minta tolong bersihkan rumah, Minta menyiapkan file, Meminjam pulpen(walaupun tahu pasti bakal hilang), Tugas dadakan dll.

Untuk itu kita harus memahami struktur dari Short Message itu sebelum kita mempraktikkannya.

Lalu bagaimana cara membuat Short Message ?

Pertama, kita harus memahami struktur secara umumnya terlebih dahulu.Struktur dalam Short Message diantaranya :

  1. Opening segment (seperti : hi, hello, dear)
  2. Purpose (tujuan)
  3. Closing segmen (seperti : thanks, thank you, love)
  4. Name sender

Setelah kita mengetahui struktur bagian umum pada Short Message.Kita juga harus mengetahui syarat-syarat Short Message dengan benar diantaranya :

  1. Close address (Kejelasan Penerima)
  2. Straight forward (isi pesan pendek, padat, jelas)
  3. Clear instruction (perintah khusus)

Nah, sekarang kalian sudah tahu tentang materi Short Message.Sekarang, kita lihat contoh text Short Message simple berikut contoh text Short Message.
